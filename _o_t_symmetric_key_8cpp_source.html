00328 /*
00329  To generate a symmetric key:
00330 
00331     1. First we generate the plain symmetric key itself using RAND_bytes().
00332     2. Then we generate the salt using RAND_bytes()
00333     3. Then we use thePassword and the salt to derive a key using PBKDF2.
00334     4. Then we encrypt the plain symmetric key using the derived key from PBKDF2.
00335     5. Then we store the salt and the encrypted symmetric key. (We discard the derived key.)
00336     6. (Use the plain symmetric key to encrypt the plaintext.)
00337 
00338  To use the symmetric key:
00339 
00340     1. We use thePassword from user input, and the stored salt, with PBKDF2 to derive a key.
00341     2. Use the derived key to decrypt the encrypted symmetric key.
00342     3. (Use the decrypted symmetric key to decrypt the ciphertext.)
00343  */
