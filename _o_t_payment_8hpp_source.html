00152 /*
00153   The PAYMENT can be of types:
00154     - CHEQUE, INVOICE, VOUCHER (these are all forms of cheque)
00155     - PAYMENT PLAN, SMART CONTRACT (these are cron items)
00156     - PURSE (containing cash)
00157 
00158  FYI:
00159  // ----------------------
00160  OTContract — Most other classes are derived from this one. Contains the actual XML contents,
00161   as well as various data values that were loaded from those contents, including public keys.
00162   Also contains a list of signatures.
00163  // ----------------------
00164  OTScriptable — Derived from OTContract, but enables scriptable clauses. Also contains a list
00165   of parties (each with agents and asset accounts) as well as a list of bylaws (each with scripted
00166   clauses, internal state, hooks, callbacks, etc.)
00167  // ----------------------
00168  OTInstrument — Has a date range, a server ID, and an asset ID. Derived from OTScriptable.
00169  // ----------------------
00170  OTTrackable  — Has a transaction number, user ID, and an asset account ID. Derived from OTInstrument.
00171  // ----------------------
00172  OTCheque — A financial instrument. Derived from OTTrackable.
00173  // ----------------------
00174  OTCronItem — Derived from OTTrackable. OT has a central “Cron” object which runs recurring tasks, known as CronItems.
00175  // ----------------------
00176  OTAgreement — Derived from OTCronItem. It has a recipient and recipient asset account.
00177  // ----------------------
00178  OTPaymentPlan — Derived from OTAgreement, derived from OTCronItem. Allows merchants and customers
00179   to set up recurring payments. (Cancel anytime, with a receipt going to both inboxes.)
00180  // ----------------------
00181  OTSmartContract — Derived from OTCronItem. All CronItems are actually derived from OTScriptable already
00182   (through OTTrackable/OTInstrument). But OTSmartContract is the first/only Cron Item specifically designed
00183   to take full advantage of both the cron system AND the scriptable system in conjunction with each other.
00184   Currently OTSmartContract is the only actual server-side scripting on OT.
00185  // ----------------------
00186  */
